# 📚 Sistema de Gestão Escolar

Sistema desenvolvido com HTML, CSS e JavaScript para gerenciamento de alunos, professores, turmas e notas.

## 🚀 Funcionalidades

- Cadastro de alunos com matrícula automática
- Cadastro de professores e disciplinas
- Criação de turmas
- Lançamento de notas
- Cálculo automático de médias
- Filtro de alunos por turma
- Persistência de dados com LocalStorage

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
